DEVICE INSURANCE ANALYSIS – Power BI Dashboard
This repository contains a Power BI dashboard that analyzes device insurance performance across different stores, months, and branches. The goal of the project is to understand attach rates, identify high‑performing locations, and highlight stores that need attention.

Business Questions
The dashboard is designed to answer questions such as:

How does the device insurance attach percentage change month by month?

Which branches and stores have the highest and lowest attach percentages?

What is the predicted attach percentage for upcoming periods?

How are stores categorized into Low, Medium, and High performance groups?

Dashboard Overview
Key visuals in the report:

Month‑wise Trend: Line chart showing attach percentage over months to track performance trend.

Branch‑wise Performance: Bar chart comparing attach percentage across branches.

Store‑level View: Table and bar visuals showing attach percentage and predicted values for each store.

Store Categorisation: Donut chart grouping stores into Low, Medium, and High categories based on attach percentage.

Slicers for Store Name, Month, and Branch allow interactive filtering.

How to Use
Download the .pbix file from this repository.

Open it in Power BI Desktop (latest version recommended).

Use the slicers on the left side to filter by month, store, or branch.

Review the insights below each visual for interpretation and business commentary.
